# Total-Resource-Optimization-System
AI-powered university timetable scheduler using a Genetic Algorithm engine, built with React, TypeScript, and Supabase Edge Functions.
A full-stack academic timetable generation system that uses a Genetic Algorithm to automatically schedule courses, faculty, and rooms while respecting hard and soft constraints.
The GA engine runs as a Supabase Edge Function and optimizes across constraints like room capacity, lab/lecture room types, faculty availability preferences, maximum teaching hours, and consecutive lab slot placement. The frontend is built with React + TypeScript and provides real-time generation progress, version history, conflict detection, and role-based access (admin only).
Key Features:

-> Genetic Algorithm with elitism, tournament selection, crossover & mutation
-> Consecutive lab pair scheduling with penalty-based fitness scoring
-> Real-time generation progress via Supabase Realtime
-> Schedule versioning with snapshot rollback
-> Faculty workload tracking and preference management
-> Role-based access control (admin-gated timetable generation)
